2. Qual formato de imagem está tomando lugar do JPG/GIF?

PNG - Portable Network Graphics

3. Qual o número máximo de cores pode ter um arquivo GIF?

256 no máximo

E JPG?

16 milhões

4. Por que não podemos utilizar o formato .gif na compactação de fotografias?

Pois este formato possui uma paleta limitada de cores (256 no máximo).

5. O que é uma GIF animada?

Blocos de controle inseridos entre imagens sucessivas determinam o tempo durante o qual cada imagem persistirá na unidade gráfica de saída, permitindo assim a apresentação seqüenciada de várias imagens.

6. O que uma GIF transparente?

Permitem especificar uma única cor que se tornará transparente, permitindo assim visualizar-se o background por entre o GIF que se definiu como transparente.
